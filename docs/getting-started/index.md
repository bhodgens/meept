# Getting Started

Get Meept installed, configured, and running your first agent task.

## Prerequisites

- **Go 1.22+** (for building from source)
- **An LLM API key** — Meept requires at least one configured LLM provider
  - Ollama (local, free) or any OpenAI-compatible API

1. [Installation](installation.md) — Build from source or install dependencies
2. [Quick Start](quick-start.md) — 5-minute walkthrough to a working agent
3. [First Run](first-run.md) — What to expect and how to verify
4. [Troubleshooting](troubleshooting.md) — Common issues and fixes
