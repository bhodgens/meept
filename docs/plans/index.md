# Plans

Archived implementation plans from the development process. These documents record design decisions, trade-offs, and implementation approaches used during Meept's development.

## Archive

The `archive/` directory contains 29 historical plan documents covering:
- Security integration and taint tracking
- Memory system improvements and vector embeddings
- LLM intent classification and model aliases
- MCP integration and tool execution
- Self-improvement and shadow training
- External integrations (Telegram, web, calendar)
- TUI enhancements (mouse scrolling, vim mode)
- Code intelligence (AST, LSP tooling)
- Agent validation and review workflows
- Bus handler deduplication and performance

See [`archive/`](archive/) for the full list.
