## Phase 1: TUI Session View Fixes

### Issue 1.1: Enter key opens detail instead of chat

**Location:** `internal/tui/models/sessions.go:220-228`

**Current behavior:**
```go
case KeyEnter:
    if len(m.sessions) > 0 {
        idx := m.table.Cursor()
        if idx >= 0 && idx < len(m.sessions) {
            m.selected = &m.sessions[idx]
            m.showingDetail = true  // Opens modal detail
        }
    }
```

**Expected behavior:** Enter should switch to chat view with that session active.

**Fix:**
```go
case KeyEnter:
    if len(m.sessions) > 0 {
        idx := m.table.Cursor()
        if idx >= 0 && idx < len(m.sessions) {
            session := m.sessions[idx]
            // Switch to chat view with session active
            return func() tea.Msg {
                return SwitchViewMsg{View: ViewChat, SessionID: session.ID}
            }
        }
    }
```

**Files to modify:**
- `internal/tui/models/sessions.go` - Update KeyEnter handler
- `internal/tui/app.go` - Handle SwitchViewMsg if not already supported

## Phase 5: Tool Panel Escape (X button + Escape key)

**User request:** "All tools that pop up something that cover something else (like the memory tool) has to be able to be escaped: put an X in the upper right and make it so hitting escape activates it"

### Issue 5.1: Add close button to full-screen tool panels

**Affected panels** (full-screen routes):
- `ui/flutter_ui/lib/features/memory/memory_panel.dart`
- `ui/flutter_ui/lib/features/settings/settings_panel.dart`
- `ui/flutter_ui/lib/features/search/search_panel.dart`
- `ui/flutter_ui/lib/features/skills/skill_panel.dart`
- `ui/flutter_ui/lib/features/projects/branches_panel.dart`

**Implementation pattern:**

```dart
class MemoryPanel extends ConsumerStatefulWidget {
  const MemoryPanel({super.key});
  @override
  ConsumerState<MemoryPanel> createState() => _MemoryPanelState();
}

class _MemoryPanelState extends ConsumerState<MemoryPanel> {
  final _focusNode = FocusNode();

  @override
  Widget build(BuildContext context) {
    return KeyboardListener(
      focusNode: _focusNode,
      onKeyEvent: (KeyEvent event) {
        if (event.logicalKey == LogicalKeyboardKey.escape) {
          context.go('/');  // Navigate back to chat tab
        }
      },
      child: Container(
        color: CyberpunkColors.black,
        child: Column(
          children: [
            // Header with X button
            Container(
              padding: const EdgeInsets.all(12),
              child: Row(
                children: [
                  const Icon(Icons.memory, color: CyberpunkColors.orangePrimary),
                  const SizedBox(width: 8),
                  Text('memory', style: ...),
                  const Spacer(),
                  IconButton(
                    icon: const Icon(Icons.close),
                    onPressed: () => context.go('/'),
                    tooltip: 'close',
                  ),
                ],
              ),
            ),
            // ... rest of panel content
          ],
        ),
      ),
    );
  }
}
```

**Files to modify:**
- `ui/flutter_ui/lib/features/memory/memory_panel.dart` - Add X button + Escape
- `ui/flutter_ui/lib/features/settings/settings_panel.dart` - Add X button + Escape
- `ui/flutter_ui/lib/features/search/search_panel.dart` - Add X button + Escape
- `ui/flutter_ui/lib/features/skills/skill_panel.dart` - Add X button + Escape
- `ui/flutter_ui/lib/features/projects/branches_panel.dart` - Add X button + Escape
