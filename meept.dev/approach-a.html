// capabilities

Built for production workloads

✹

Multi-Agent Orchestration

Specialist agents for coding, debugging, planning, and analysis — dispatched and routed by a central classifier. Agents collaborate via channel-based pairing.

⭐

5-Tier Memory System

From episodic recollections to task-level working memory with domain FTS5 search. All backed by SQLite with automatic deduplication and consolidation.

☑

Evidence Pipeline

Every agent action is logged, verified, and auditable. Security engine checks permissions pre-execution. Full audit trail with SQLite-backed records.

●

Context Firewall

Intelligent context compaction with summary generation, message dropping, and pressure monitoring. Stats expose drop rates and compaction savings in real time.

⛐

Skill-Based Routing

Three-tier skills discovery with priority shadowing. YAML frontmatter declares requirements; cheapest compliant model gets auto-selected for each task.

⌶

Q Agent — Self-Optimization

A meta-agent that analyzes sessions for improvement opportunities, generates optimizations, and applies them autonomously — the system that writes itself.

$ make build

$ ./bin/meept chat "write a unit test"
