Self-Hosted Daemon

One binary. No cloud dependency. Runs on your laptop, CI runner, or bare metal. Your prompts, your code, your data — never in someone else's sinkhole.

Episodic Memory

Every interaction persists in SQLite-backed FTS5 indexes. Agents remember context across sessions, consolidate learnings, and build institutional knowledge. No cold starts.

memory> find --semantic "postgres connection pool" 3 matches across 12 sessions → pr/482: connection timeout fix (coder) → session/291: pool tuning discussion (analyst) → session/204: initial db setup (planner) memory> consolidate --domain db-optimization 3 facts merged into task memory

Security Engine

Tirith-based command scanning, TLS everywhere, SQLite permission matrix, tool gating with granular ACLs, and full audit logging. Your agents can be powerful without being reckless.

Three commands.

Running in 60 seconds.

bash 
# 1. Install
$ curl -fsSL https://meept.dev/install.sh | bash

# 2. Start the daemon
$ go build -o bin/meept-daemon ./cmd/meept-daemon
$ ./bin/meept-daemon -f
 ✓ daemon running · pid 48271 · 8 agents

# 3. Chat
$ ./bin/meept chat "Migrate our connection pool from pgx to lib/pq"
 → dispatching to coder + planner...
 → planner decomposed into 3 tasks
 → executing: analyze current pool config
 → complete. 14 files changed. 23 tests pass.
