02
Core capabilities

Features

Multi-Agent Orchestration

Specialized agents (coder, debugger, planner) route tasks intelligently via intent classification.

Memory & Context

SQLite-backed episodic and task memory with FTS5 search, retention policies, and automatic summarization.

Security Engine

Multi-layer security: prompt injection filtering, tool gating, shell scanning via Tirith, and TLS enforcement.

Skill Registry

YAML-defined skills with capability-based model resolution. Three-tier priority: project, user, system.

Self-Improvement

Automated detection, analysis, generation, and validation of workflow improvements in a full feedback loop.

Multi-Transport

Connect via Unix socket RPC, REST API, WebSocket, or Telegram — same backend, any interface.

04
Get started in 30 seconds

Quick Start

# install the daemon
$ go install github.com/caimlas/meept/cmd/meept-daemon@latest

# start the service
$ meept-daemon -f

# chat via CLI
$ meept chat "build me a REST API"
